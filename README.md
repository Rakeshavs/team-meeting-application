# Shnoor Meetings Project

A professional, full-stack video meeting platform integrated with Supabase and Render.

## Project Structure

This project is organized into two main parts:

### 1. [Frontend](./frontend)
The React/Vite web application.
- **Location**: `/frontend`
- **To Run**:
  1. `cd frontend`
  2. `npm install`
  3. `npm run dev`

### 2. [Backend](./backend)
The FastAPI signaling and signaling server.
- **Location**: `/backend`
- **To Run**:
  1. `cd backend`
  2. `python main.py`

## Features
- **Video Calls**: Real-time WebRTC communication.
- **Supabase Integration**: Unified user profiles and cloud sync.
- **Theme Support**: Premium UI/UX color themes throughout.
- **Calendar**: Meeting scheduling and history.

## Deployment
This project is configured for one-click deployment to **Render.com**.
1. Push your branch to GitHub.
2. Connect the repository to Render.
3. Set your environment variables as detailed in the deployment guide.
